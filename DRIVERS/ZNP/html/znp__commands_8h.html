Detailed Description

Simple definitions of ZNP commands.

These are listed MSB first, as the bytes come "down the wire" instead of LSB first like everything else.

Note:
you may want to print this out and keep it handy, as it is very useful when debugging.
this is not a comprehensive list of all ZNP commands, only what is currently used by the library or seen when running the examples.
